# java 对象结构小结

### 1. 权限修饰符

`default` `public` 可以修饰类。
`private` `protected` 可以修饰类和成员。
**静态成员不属于对象，而是属于类对象**。

#### 2. 对象结构说明

- **头部**
1. Mark Word记录了对象和锁有关的信息，当这个对象被synchronized关键字当成同步锁时，围绕这个锁的一系列操作都和Mark Word有关。
2. 指向 Class对象的指针
3. 数组长度（只有数组对象才有）

- **继承**
  所有的父类的成员。
  ```java
  private int a =1;
  protected int b =1;
  public double c =1.0;
  ```
- **自身**
  ```java
  public int a =1;
  private int d =1;
  public int f=2;
  ```
- **对齐填充字节**  
JVM要求java的对象占的内存大小应该是8bit的倍数，所以后面有几个字节用于把对象的大小补齐至8bit的倍数，没有特别的功能。
