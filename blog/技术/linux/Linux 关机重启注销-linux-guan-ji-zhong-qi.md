# 关机重启
shutdown -h now 立刻进行关机
shutdown -h 1      1分钟后关机
shutdown -r now 现在重启计算机
shutdown -c 取消关机
halt 关机
reboot 现在重启计算机
sync 把内存的数据同步到磁盘
![image-1656293415639](/upload/2022/06/image-1656293415639.png)
> 直接输入shutdown，默认是相当于输入shutdown -h 1
> 也就是1分钟后关机

# 注销
在terminal里输入logout可以注销用户
> logout在图形运行级别无效果，在运行级别3下有效
